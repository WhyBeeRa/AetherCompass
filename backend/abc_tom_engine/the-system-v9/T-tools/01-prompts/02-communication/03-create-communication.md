# Prompt: Create Professional Communication

Copy and paste this into your chat (fill in the details first):

---

```
I need to write a professional communication. Here's the situation:

---
Type (pick one): Email / Slack message / Internal update / Client proposal / Community message / HR communication / Other
Recipient: [Who is this for? Name, role, company]
What they need: [What problem are they trying to solve? What did they ask for?]
What I'm proposing/communicating: [Your solution, update, request, or message]
Budget/pricing (if relevant): [Price range, package details, or "not applicable"]
Context: [How did this come up? Any previous conversations? What do they already know?]
---

Adam, please manage this project:

1. Pre-flight check: Read C-core/project-brief.md, C-core/voice-dna.md, and C-core/icp-profile.md. If any of these files are empty or still have placeholder text, STOP and tell me which files need to be filled first.

2. Read the workflow in T-tools/03-workflows/communication-workflow.md to understand the full process

3. Read my communication samples: B-brain/02-my-samples/02-communication/

4. Read the relevant skill from T-tools/02-skills/:
   - Client proposal or sales email → sales-proposal-skill
   - HR or internal communication → internal-communication-skill
   - Professional email → professional-email-skill

5. Read past learnings: M-memory/learning-log.md

6. Now delegate to the Strategist (read A-agents/strategist-agent.md first):
   - Analyze the recipient's situation and real needs
   - Map the opportunity and positioning
   - Identify what makes this communication compelling for THIS specific recipient
   - Create a new folder in O-output/ with the next available number, and a _process/ subfolder inside it
   - Save as _process/strategist-analysis.md

7. Review the Strategist's work, then delegate to the Copywriter (read A-agents/copywriter-agent.md first):
   - Write the communication in MY voice and style
   - Use the Strategist's analysis to inform the approach
   - Apply the right skill format for the chosen type
   - Save as _process/copywriter-draft.md with notes about choices made

8. Send to the Gatekeeper (read A-agents/gatekeeper-agent.md first):
   - Review for persuasiveness, clarity, and voice match
   - Check: Would I actually send this? Does it sound like me?
   - Save as _process/gatekeeper-review.md
   - If approved, create final-communication.md at the PROJECT FOLDER ROOT (not inside _process/)
   - If not approved, manage revisions

9. Update M-memory/learning-log.md with what you learned from this process

Show me the final result and tell me if it's ready to send or needs revision.
```
