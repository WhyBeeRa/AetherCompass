## What to Do

Follow the content workflow exactly:
1. **Ask** what type of content (social post / blog post / newsletter) and the topic
2. **Researcher** finds angles, data, and supporting material
3. **Copywriter** writes the draft in the owner's voice using the relevant skill
4. **Gatekeeper** reviews quality, voice match, and standards
5. If revisions needed, Copywriter revises and Gatekeeper reviews again
6. Save the final output to a project folder in O-output/
7. Update M-memory/learning-log.md

**NEVER use em dashes or en dashes.** Use periods, commas, or colons instead.

**Language:** Respond in the language the user uses (Hebrew or English)
