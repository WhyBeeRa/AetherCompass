# Start Here / התחילו כאן

## English

Welcome to the ABC-TOM system (v9).

**Step 1:** Open the workshop instructions file:
`@ALL-GUIDES-HERE/01-workshop-exercises/Workshop Exercises (Step by Step).md`

It has step-by-step instructions for the entire workshop.

**Step 2:** Open Claude Code and set this folder (the-system) as your working directory.

**Step 3:** Choose your track:
- **Content** (תוכן) - publishable content: social posts, newsletters, blog posts
- **Communication** (תקשורת) - professional text to a recipient: emails, internal messages, community, education
- **Thinking** (חשיבה) - decision briefs: bring a decision, multiple agents debate it, you get a brief with different perspectives

The instructions file will guide you through everything.

**Bonus guides:**
- `05-quick-start-by-role/` - Find your role, get your first output in 10 minutes
- `06-what-can-i-build/` - Real examples of what you can create, organized by profession
- `07-core-examples/` - Filled-in examples for different roles
- `08-system-map/` - How the entire system works, in one page

## עברית

ברוכים הבאים למערכת ABC-TOM (v9).

**שלב 1:** פתחו את קובץ ההוראות:
`@ALL-GUIDES-HERE/01-workshop-exercises/הוראות לתרגילי הסדנה (שלב אחרי שלב).md`

בפנים יש הוראות שלב-אחרי-שלב לכל הסדנה.

**שלב 2:** פתחו את Claude Code ובחרו את התיקייה הזאת (the-system) כתיקיית העבודה.

**שלב 3:** בחרו את המסלול שלכם:
- **תוכן** (Content) - תוכן לפרסום: פוסטים, ניוזלטרים, מאמרים לבלוג
- **תקשורת** (Communication) - טקסט מקצועי לנמען: מיילים, הודעות סלאק, עדכונים פנימיים, קהילה, חינוך
- **חשיבה** (Thinking) - תדריכי החלטות: הביאו החלטה, כמה סוכנים מתווכחים עליה, ותקבלו תדריך עם נקודות מבט שונות

קובץ ההוראות ידריך אתכם בכל השאר.

**מדריכים נוספים:**
- `05-quick-start-by-role/` - מצאו את התפקיד שלכם, קבלו תוצאה ראשונה ב-10 דקות
- `06-what-can-i-build/` - דוגמאות אמיתיות למה שאפשר ליצור, מאורגן לפי מקצוע
- `07-core-examples/` - דוגמאות מלאות לתפקידים שונים
- `08-system-map/` - איך כל המערכת עובדת, בעמוד אחד
